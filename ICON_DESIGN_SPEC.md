# IconStore 图标绘制规范

## 1. 画布与网格

| 属性 | 规范 |
|------|------|
| **画布尺寸** | 24 × 24 px |
| **安全区域** | 内容控制在 22 × 22 区域内，保留 1px 边距 |
| **网格** | 使用 1px 网格，对齐像素 |
| **背景** | 透明（无底色） |

```
┌──────────────────────┐
│  ▫ ▫ ▫ ▫ ▫ ▫ ▫ ▫ ▫  │  ← 安全区域边距 1px
│  ▫ ┌──────────────┐ ▫ │
│  ▫ │              │ ▫ │
│  ▫ │  图标内容区域   │ ▫ │
│  ▫ │  22 × 22      │ ▫ │
│  ▫ │              │ ▫ │
│  ▫ └──────────────┘ ▫ │
│  ▫ ▫ ▫ ▫ ▫ ▫ ▫ ▫ ▫  │
└──────────────────────┘
```

## 2. 线性风格规范 (Linear)

| 属性 | 规范 |
|------|------|
| **描边宽度** | **1.5px**（固定值，不可使用其他宽度） |
| **端点样式** | `stroke-linecap="round"`（圆头端点） |
| **连接样式** | `stroke-linejoin="round"`（圆角连接） |
| **填充** | `fill="none"` |
| **颜色** | `stroke="currentColor"` |
| **路径交叉** | 避免不必要的路径交叉和重叠 |

### 线性样式示例

```
✅ 正确                              ❌ 错误
┌──────────────┐                    ┌──────────────┐
│    ╭────╮    │                    │    ┌────┐    │
│    │    │    │                    │    │    │    │
│    ╰────╯    │                    │    └────┘    │
│  stroke=1.5  │                    │  stroke=2.0  │
│  round端点    │                    │  butt端点     │
│  round连接    │                    │  直角连接      │
└──────────────┘                    └──────────────┘
```

## 3. 面型风格规范 (Filled)

| 属性 | 规范 |
|------|------|
| **填充** | `fill="currentColor"` |
| **描边** | 无描边（不使用 stroke） |
| **镂空区域** | 使用 path 的奇偶填充规则（evenodd）实现镂空 |
| **透明度** | 不使用透明度，纯色填充 |

### 面型样式示例

```
线性版本                        面型版本
┌──────────────┐               ┌──────────────┐
│    ╭────╮    │               │    ▓▓▓▓▓▓    │
│    │    │    │               │    ▓▓▓▓▓▓    │
│    ╰────╯    │               │    ▓▓▓▓▓▓    │
│  轮廓线条     │               │  实心填充     │
└──────────────┘               └──────────────┘
```

## 4. 圆角规则

| 元素类型 | 圆角值 | 说明 |
|---------|--------|------|
| **外轮廓** (矩形/圆角矩形) | **2px** | 例如对话框、卡片图标 |
| **内部结构** | **1px** | 例如线条转折处、小形状 |
| **圆形/椭圆** | 100% | 自然圆形无需额外设置 |
| **点状元素** | 100% | 小于 4px 的小圆点 |

## 5. 间距与留白

```
┌──────────────────────┐
│                      │
│   ╭──────────╮       │
│   │          │       │  ← 图标主体距容器边 ≥ 1px
│   │          │       │
│   ╰──────────╯       │
│                      │
└──────────────────────┘
```

| 场景 | 最小间距 |
|------|---------|
| 图标内容与 24×24 边缘 | ≥ 1px |
| 内部元素之间的间距 | ≥ 2px |
| 线条与线条之间的间隙 | ≥ 2px |

## 6. 视觉重量平衡

**原则**：所有图标在视觉上应该是「一样重」的。

| 形状 | 实际应占尺寸 | 说明 |
|------|-------------|------|
| 方形图标（如 `stop`） | 18-20px | 方形视觉偏重，应适当缩小 |
| 圆形图标（如 `play`） | 20-22px | 圆形视觉偏轻，可以稍大 |
| 水平长条（如 `menu`） | 18-20px | 水平方向视觉偏重，适当缩短 |
| 垂直长条（如 `up-down`）| 20-22px | 垂直方向偏轻，可适当拉长 |
| 复杂图标 | 20-22px | 细节多的图标需要更多空间 |

## 7. 线性 → 面型 转换规则

两个风格的图标必须**结构一致**，只是视觉表现不同：

```
线性                         面型
┌──────────┐               ┌──────────┐
│  ╭─╮     │               │  ▓▓▓     │
│  │ │     │    ─────▶     │  ▓▓▓     │
│  ╰─╯     │               │  ▓▓▓     │
│  轮廓描边  │               │  实心填充  │
└──────────┘               └──────────┘
```

| 属性 | 线性 | 面型 |
|------|------|------|
| 形状 | 完全一致 | 完全一致 |
| 线条 | 1.5px 描边 | 无描边 |
| 填充 | 无填充 | `currentColor` |
| 镂空 | 空区域 | `fill-rule="evenodd"` |

## 8. 命名规范

```
格式: 类别-具体-描述    (全小写连字符)
示例: media-radio        ✓
     arrow-left-up      ✓
     user-profile-edit  ✓
     2k-files           ✗ (不要数字开头)
     MediaRadio         ✗ (不要驼峰)
     media_radio        ✗ (不要下划线)
```

命名即翻译！`iconLabel.ts` 会自动按 `-` 拆词翻译。

## 9. Figma 导出设置

| 设置 | 值 |
|------|-----|
| 导出格式 | SVG |
| 视图框 | 包含图层边界 |
| 简化路径 | 开启 |
| 轮廓化描边 | 关闭（保留 stroke 属性） |
| 颜色 | currentColor |
| 小数位数 | 最多 2 位小数 |

## 10. 设计红线

❌ **不要使用多个颜色** — 所有图标只使用 `currentColor`，跟随文本颜色
❌ **不要使用渐变** — 图标不支持渐变效果
❌ **不要使用阴影** — 图标本身不带阴影
❌ **不要使用透明度** — 纯色填充，不使用 opacity
❌ **不要使用多种描边宽度** — 线性风格统一 1.5px

✅ **保持简洁** — 最少路径数实现可识别的图形
✅ **风格统一** — 所有图标看起来像是同一双手画出来的
✅ **语义清晰** — 图标的意义应该能直观理解，不需要文字说明

## 11. 导出 SVG 模板

### 线性版
```svg
<svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round">
  <path d="..." />
</svg>
```

### 面型版
```svg
<svg viewBox="0 0 24 24" fill="currentColor">
  <path d="..." />
</svg>
```

## 12. 检查清单

绘制完成后，对照以下清单检查：

- [ ] viewBox 为 `0 0 24 24`
- [ ] 线性版 stroke-width 为 1.5
- [ ] 线性版 stroke-linecap 为 round
- [ ] 线性版 stroke-linejoin 为 round
- [ ] 线性版 fill 为 none
- [ ] 面型版无 stroke 属性
- [ ] 所有颜色使用 currentColor
- [ ] 不使用透明度、渐变、阴影
- [ ] 不使用多种描边宽度
- [ ] 两个风格版本结构一致
- [ ] 命名全小写连字符格式
